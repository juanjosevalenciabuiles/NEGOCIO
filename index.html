Conectar Supabase
Supabase es gratuito. Datos en la nube, accesible desde cualquier dispositivo.
1
Ve a supabase.com, crea cuenta y proyecto.
2
En SQL Editor ejecuta este código:
drop table if exists movimientos_recarga cascade; drop table if exists clientes_recarga cascade; create table clientes_recarga ( id uuid default gen_random_uuid() primary key, nombre text not null, telefono text, plataforma text, proveedor text, codigo_cliente text, nombre_whatsapp text, deuda_anterior numeric default 0, cupo_maximo numeric default 0, creado_en timestamptz default now() ); create table movimientos_recarga ( id uuid default gen_random_uuid() primary key, cliente_id uuid references clientes_recarga(id) on delete cascade, tipo text check (tipo in ('entrega','pago')), valor numeric not null, fecha date default current_date, hora time, codigo_transaccion text, codigo_cliente text, codigo_cuenta text, comprobante text, nota text, confirmado boolean default false, confirmado_en timestamptz, creado_en timestamptz default now() ); alter table clientes_recarga add column if not exists nombre_whatsapp text; alter table clientes_recarga disable row level security; alter table movimientos_recarga disable row level security;
3
Ve a Project Settings → API. Copia la Project URL y la clave anon public.
