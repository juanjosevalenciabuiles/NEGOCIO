Conectar Supabase
Supabase es gratuito. Los datos quedan en la nube y puedes entrar desde cualquier dispositivo.
1
Ve a supabase.com, crea cuenta y proyecto nuevo.
2
En SQL Editor ejecuta este código:
create table clientes_recarga ( id uuid default gen_random_uuid() primary key, nombre text not null, telefono text, plataforma text, proveedor text, codigo_cliente text, deuda_anterior numeric default 0, cupo_maximo numeric default 0, creado_en timestamptz default now() ); create table movimientos_recarga ( id uuid default gen_random_uuid() primary key, cliente_id uuid references clientes_recarga(id) on delete cascade, tipo text check (tipo in ('entrega','pago')), valor numeric not null, fecha date default current_date, hora time, codigo_transaccion text, codigo_cliente text, codigo_cuenta text, comprobante text, nota text, confirmado boolean default false, confirmado_en timestamptz, creado_en timestamptz default now() ); alter table clientes_recarga disable row level security; alter table movimientos_recarga disable row level security;
3
Ve a Project Settings → API. Copia la Project URL y la clave anon public.
