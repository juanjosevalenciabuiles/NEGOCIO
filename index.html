Conectar base de datos (Supabase)
Supabase es gratuito. Los datos quedan guardados en la nube y puedes entrar desde cualquier dispositivo.
1
Ve a supabase.com, crea una cuenta gratuita y un proyecto nuevo.
2
En tu proyecto abre SQL Editor y ejecuta este código:
create table clientes_recarga ( id uuid default gen_random_uuid() primary key, nombre text not null, telefono text, plataforma text, proveedor text, deuda_anterior numeric default 0, cupo_maximo numeric default 0, creado_en timestamptz default now() ); create table movimientos_recarga ( id uuid default gen_random_uuid() primary key, cliente_id uuid references clientes_recarga(id) on delete cascade, tipo text check (tipo in ('entrega','pago')), valor numeric not null, fecha date default current_date, comprobante text, nota text, creado_en timestamptz default now() ); create table prestamos ( id uuid default gen_random_uuid() primary key, nombre text not null, telefono text, capital numeric not null, tasa_interes numeric not null, total_abonos numeric default 0, meses_pagados integer default 0, fecha_inicio date default current_date, garantia text, nota text, creado_en timestamptz default now() ); create table movimientos_prestamo ( id uuid default gen_random_uuid() primary key, prestamo_id uuid references prestamos(id) on delete cascade, tipo text check (tipo in ('interes','abono_capital')), valor numeric not null, fecha date default current_date, comprobante text, nota text, creado_en timestamptz default now() );
3
Ve a Project Settings → API. Copia la Project URL y la clave anon public.
4
Pégalas aquí abajo y haz clic en Guardar y conectar.
